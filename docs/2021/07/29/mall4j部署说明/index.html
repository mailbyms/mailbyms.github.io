遇到的问题

Q: UniAPP 客户端注意账号时数据库报错

### Cause: com.mysql.cj.jdbc.exceptions.MysqlDataTruncation: Data truncation: Data too long for column ‘login_password’ at row 1

A： tz_user 表的 login_password 只有 50字节，改为 128 字节

三、部署 Redis
